Getting Started:

To use the API is really easy! Currently v.1 supports 1 endpoint. This will gather all information about a Teespring campaign.

HTTP Request

GET: http://teespring-api.cloudapp.net/v01/[campaign-name]

Right now, the only query parameter supported are teespring campaign names, just throw it at the end and you're good to go!

Expected output:


  {  
   "tee":"[campaign-name]",
   "url":"[campaign-url]",
   "images":{  
      "front":"[front-image-url",
      "back":"[back-image-url]"
   },
   "total_sold":"[amount-currently-sold",
   "goal_amount":"[goal-amount]",
   "goal_date":"[goal-end-date]",
   "details":"[campaign-details]"
}

Here is a simple Node.JS example:


var request = require('request');
request('teespring-api.cloudapp.net/v01/tudev', function (error, response, body) {
  if (!error && response.statusCode == 200) {
    JSON.parse(body);
  }
})
