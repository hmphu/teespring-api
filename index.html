Getting Started:

To use the API is really easy! Currently v.1 supports 1 endpoint. This will gather all information about a Teespring campaign.

Here is a simple request Node.JS example:


var request = require('request');
request('teespring-api.cloudapp.net/tudev', function (error, response, body) {
  if (!error && response.statusCode == 200) {
    JSON.parse(body);
  }
})
